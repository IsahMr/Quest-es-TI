2. Qual a diferença entre um site estático e dinâmico?

Site estático: O conteúdo permanece o mesmo para todos os usuários e não muda com base em interações do usuário. Site dinâmico: O conteúdo é gerado ou modificado com base nas interações do usuário, podendo variar de acordo com as ações dos visitantes.

3. Para que os programadores utilizam o Github?

Os programadores utilizam o Github para hospedar projetos de software, colaborar com outros desenvolvedores, revisar e controlar versões de código, além de rastrear problemas e solicitações de recursos.

4. O que significa o acrônimo C.R.U.D?

C.R.U.D significa Create (Criar), Read (Ler), Update (Atualizar) e Delete (Excluir). É uma abreviação que representa as operações básicas de manipulação de dados em bancos de dados ou sistemas de informação.

5. Qual é a função da linguagem SQL?

A função da linguagem SQL é controlar o banco de dados.

6. O que é A.P.I?

O A.P.I (Application Programming Interface) ou Interface de Programação de Aplicação que facilita a comunicação e troca de informações entre sistemas.

7. Qual é a função do CSS?

A função do CSS é de modificar a aparência do HTML.

8. O que significa "responsividade" em páginas WEB?

A responsividade de uma página WEB é adaptar o tamanho das suas páginas (alteração do layout) ao tamanho das suas telas que estão sendo exibidos.

9. Qual o papel do U.X. (User Experience)?

O papel do U.X é garantir que a interação entre os usuários e um produto digital seja intuitiva, eficiente e agradável.

10. O que é um wireframe em páginas WEB?

O wireframe seria o esboço da página, que pode ser feito em um papel ou digitalmente.
